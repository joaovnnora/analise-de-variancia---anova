# Análise de Variância (ANOVA)

Este projeto apresenta um estudo acadêmico sobre os fundamentos da Análise de Variância (ANOVA). O foco principal é a exploração teórica dos cálculos estatísticos e a sua implementação em linguagem Python.

## 1. Objetivos
O projeto detalha a decomposição da variância total em componentes explicados (entre grupos) e não explicados (dentro dos grupos). O objetivo é compreender como a estatística F é construída a partir dessas variações para testar a igualdade de médias entre diferentes populações.

## 2. Teoria
O conteúdo aborda os pilares matemáticos da ANOVA:
- Definição de Hipótese Nula (H0) e Hipótese Alternativa (H1).
- Cálculo da Soma dos Quadrados (Total, Entre Grupos e Residual).
- Determinação dos Graus de Liberdade e aplicação do Teste F de Snedecor.
- Análise do P-valor para tomada de decisão estatística.

## 3. Exemplo
O notebook inclui a resolução de um exemplo prático extraído diretamente da literatura de referência, demonstrando a aplicação dos conceitos teóricos e a validação dos resultados via Python.
